## ВАЖНО!!! Перед первым запуском софта необходимо:

- убедиться в правильности активации виртуального окружения VENV

- установить библиотеки командой: pip install -r requirements.txt

- расположить файл `accounts.xlsx` должен находиться в папке config/data/...

- расположить файл `.env` должен находиться в папке config/...

## ВАЖНО!!! Все софты в новых проектах постоянно обновляются (бывает, в день по несколько раз)... 
## Поэтому перед запуском того, либо иного софта, всегда необходимо делать обновление "Master > Update Project"!
## Если на момент обновления софта у вас уже запущены активности в нём, то обновление к ним не применится и активности нужно будет перезапустить в PowerShell.



## Старт 02.03.26:

- `MidenWalletCreate.py` - создание Miden Wallet по сид-фразе MetaMask
    - ссылка на расширение для ADS https://chromewebstore.google.com/detail/miden-wallet/ablmompanofnodfdkgchkpmphailefpb
    - стоит фильтр на успешную регистрацию
    - в случае возникновения ошибки, удалите расширение в профиле, закройте профиль и перезапустите софт

- `NativeTokenFaucetPublic.py` (ADS) - кран нативного токена https://faucet.testnet.miden.io/
    - количество минта токенов: рандом 1-3
    - каждый минт - рандомная сумма (100, 500 или 1000 $MIDEN)
    - скрипт для получения в формате Public Note
    - фильтр на 24 часа
    - фильтр на общее количество минтов (по умолчанию 9)
    - ВАЖНО!!! Не стоит злоупотреблять данным скриптом - достаточно 1-3 запусков (далее по необходимости).

- `NativeTokenFaucetPrivate.py` (ADS) - кран нативного токена https://faucet.testnet.miden.io/
    - количество минта токенов: рандом 1-3
    - каждый минт - рандомная сумма (100, 500 или 1000 $MIDEN)
    - скрипт для получения в формате Private Note
    - фильтр на 24 часа
    - фильтр на общее количество минтов (по умолчанию 3)
    - ВАЖНО!!! Данный скрипт достаточно запустить один раз.
